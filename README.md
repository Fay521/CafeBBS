# CafeBBS — 咖啡论坛 Android 客户端

LightCafe 社区（bbs.liht.cc）的 Android 客户端，基于 [Flarent](https://github.com/zxbmmmmmmmmm/Flarent-Android) 二次开发。

## 功能

- 📋 浏览讨论帖和帖子
- 🏷️ 分类标签浏览
- ✉️ 私信对话
- 🔔 实时推送通知
- 🗳️ 投票（fof/polls）
- 🎭 表情回应（fof/reactions）
- 📝 Markdown 编辑器 + 预览
- 🎮 经验等级系统
- 👤 个人资料编辑
- ☕ 咖啡厅定制主题
- 📎 文件上传
- 🔗 社交频道入口（QQ群 / TG / Discord）

## 下载

[GitHub Releases](https://github.com/Fay521/CafeBBS/releases)

## 构建

1. 修改 `app/src/main/res/values/forum_config.xml` 中的配置
2. 使用 Android Studio 打开项目，Build → Build APK

或命令行：
```bash
./gradlew assembleDebug
```
